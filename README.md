# Warpspace Chrome Extension

This is a Chrome extension  I

## Commands

# Default

- New tab
- New window
- ... commands from window context

# Tab

- Switch to open tab
- Mute/unmute
- Pin/unpin
- Move to window
- Copy to window
- Close

# Page

- Open in new tab
- Add to window...
  - Search / new window
- Open in this tab
- Favorite
- Add to saved window

# Window

- Rename window
- Close window
